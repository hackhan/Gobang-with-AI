Gobang_with_AI
=====
一个带有简易 AI 的五子棋游戏，界面基于开源图形库 ege 编写(https://github.com/misakamm/xege)

AI 采用了极大极小搜索和估值函数。

规则简介：
-----------------------------------

        1. 采用 15 × 15 的棋盘
        2. 采用指定开局规则，由黑方从 26 种开局中任选一种
        3. 白方有“三手可交换”权
        4. 黑方行驶五手 N 打规则
        5. 黑方有三三禁手、四四禁手、长连禁手

目前存在的问题：
-----------------------------------

        1. 极大极小搜索只能深入一层
        2. α-β 剪枝没有实现
        3. 禁手规则尚未实现
